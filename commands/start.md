Start this session:

1. Pull the latest from `main` and make sure we're up to date.
2. Ask me what we're building today (one sentence is fine).
3. Based on my answer, create a feature branch with a short, descriptive name (`feature/short-name` or `fix/short-name`).
4. Read CLAUDE.md and summarize: project context, current Active Work items, and any recent decisions in the log that might affect today's work.
5. Confirm we're ready, then wait for instructions.

**Working agreement for this session:**
- Make small, atomic commits — one logical change per commit
- Commit as we go, not all at the end
- Flag concerns proactively rather than working around them
- Stay focused on this one task
